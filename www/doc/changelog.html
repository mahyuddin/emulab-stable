Changelog/Technical Details

August 26, 2002

Added support for uploading SSH protocol 2 public keys via the web interface. Previously, only protocol 1 keys were allowed. Also added support for converting SSH2 (SSH Communications) and SECSH public key files into OpenSSH key format, which is what Emulab uses internally. This should solve the problems people have reported with their public keys being rejected by Emulab!
September 17, 2002

Added support for 'short names' in /etc/hosts files for nodes that are not directly connected, but are in the node's routing table. See our new faq entry on the subject of node/interface naming for details.
